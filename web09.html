JS课堂第一节 - JavaScript 基础入门

分类：WEB前端 发布时间：2017-04-18 11:09 作者：Vlin 来源：Vlin.me

JavaScript诞生于1995年，为表单验证而生。 JavaScript 作为一种脚本语言，被广泛的应用于浏览器端和服务端。在这一部分，我们将对 JavaScript 有一个简单的介绍，并在之后对 JavaScript 中的基本输出方式、变量相关定义和使用进行学习。

JavaScript 简介

    1. 为什么要学？
  1. 提高用户体验
  2. 增加动效
  3. 开发强大的WEB应用
    2. 能做什么？
  1. 增强页面动态效果（如：下拉菜单、图片轮播、信息滚动等）
  2. 实时、动态交互（如：表单验证等）
  3. Ajax网络请求数据
    3. 组成部分？

    Javascript是一种专为网页交互而设计的脚本语言，由下列三个不同的部门组成：

  1. 1.ECMAScirpt， 由ECMA-262定义，提供核心语言功能，js核心语言标准；
  2. 2.文档对象模型（DOM），提供访问和操作网页内容的方法和接口；
  3. 3.浏览器对象模型（BOM），提供与浏览器交互的方法和接口。
    4. 使用方式？
  1. 4.1 JavaScript的引入方法一：内嵌代码
  2. 4.2 JavaScript的引入方法二：外部引入 优点：可维护性、可缓存、适应未来
  3. 4.3 JavaScript的引入方法三：延迟脚本

变量

    1. 啥叫变量？

变量是一个储存数值的容器；

    2. 游戏规则

    【变量命名】变量可以用字母、 数字以及下划线（_）或者美元符（$） 组成。

  1. 1.必须以字母、下划线或美元符号开头，后面可以跟字母、下划线、美元符号和数字。
  2. 2.变量名区分大小写，如：myvar 与 myVar 是两个不同的变量。
  3. 3.不允许使用 JavaScript 关键字和保留字作为变量名，比如说break， Boolean

    4. 数据类型
  1. String
  2. Number
  3. Boolean
  4. Array
  5. Object
  6. undefined和null
